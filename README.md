A WIP addon for Create: Northstar - Redux that fully overhauls the lunar terrain, with new surface features, including craters of all sizes, scattered moon rocks, and basaltic maria.
Builds can be found [here](https://github.com/Doodlekittie/Northstar-Lunar-Terrain/releases).
Currently, the mod has most of its features and is in a fully playable state. The only things missing are carver caves, which will connect from the surface to the cave layer at y = -10. For now, you’ll just have to dig.
The other missing feature is rilles in the maria, which serve no functional purpose but will look cool.
